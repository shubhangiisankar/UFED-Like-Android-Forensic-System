# UFED-Like-Android-Forensic-System

This project is a simplified Android forensic analysis system inspired by UFED tools. It performs logical acquisition, deleted file recovery, metadata extraction, hashing, and forensic report generation.

## Features
- Android device detection using ADB
- Logical data acquisition from internal storage
- File system parsing
- Deleted video recovery
- Metadata extraction
- MD5 and SHA256 hash generation
- Automated forensic report generation

## Technologies Used
- Python
- Android Debug Bridge (ADB)
- Digital Forensics Concepts

## Output
Recovered files and forensic reports are generated for analysis.
